Stage 1: Immediate counterexample sweep
- Run sanity checks on boundary cases, degenerate parameter regimes, dimensional edge cases, and sign/domain constraints.

Stage 2: First proof attempt
- Attempt a direct proof using the current assumptions only.

- If none are found, continue to Stage 4.

Stage 4: Deep proof re-attempt (mandatory deep pass)
- Rebuild a proof strategy using insights from Stage 3 failures.
